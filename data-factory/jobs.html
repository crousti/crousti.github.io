Product-Live Data Factory

Jobs

Product-Live Data Factory allows you to create Jobs. A Job allows you to create imports and exports of data stored in Product-Live or manipulate files and data that are stored outside of Product-Live.
These Jobs consist of Tasks that are chained together. Tasks are elementary actions, for example: FTP Get (allows to get a file on a sFTP/FTP server), CSV to XML (allows to convert a CSV file to a XML file which is easier to manipulate)...
The full list of elementary Tasks can be found here: view all Tasks.

Jobs can be executed manually or periodically.

An example of creating a Job for daily import data from your ERP to Product-Live would be as follow:

  1. FTP Get Get the latest CSV file on a FTP server.
  2. CSV to XML Transform the CSV to a XML format.
  3. Transform XLST Transform the XML to the format expected by Product-Live
  4. Import Items Import products to Product-Live

In this Job there are 4 Tasks chained.

Pipelines

Jobs are executed in Pipelines. A pipeline is a work stack of Jobs with rule: first in first out.
By default you have one Pipeline on your account, but you can purchase as many Pipelines as you want. For example you can need multiple Pipelines in these cases:

  • You want to execute Jobs in parallel.
  • You want that the Jobs of your Sandbox Space can not have slowdown effects on Jobs of your Production Space.
  • You need that a Job must be executed at a precise time.
If you have several Pipelines, then a job can be associated with one or multiple Pipelines. This allow you to reduce the cost of adding Pipelines and find the right number of Pipelines for your needs.

Create and edit Jobs

To create these Jobs you can choose between creating them with the graphical user interface or with an advanced mode where you describe your Job in a JSON file. First go to:

  1. Product-Live Settings

  3. Data Factory

  5. Jobs

The graphical user interface

The graphical user interface is a good way to start. This is simple, even a non technical user can use this interface.

The advanced edition in JSON

The advanced edition is prefered by the advanced users of Product-Live Data Factory. It allows to quickly duplicate or mass edit Jobs definition.
For this you can download the VS Code Extension that will help you a lot for creating a valid JSON description of your Job.

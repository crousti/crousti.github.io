Generate Excel

You need to to generate one or multiple .xlsx file. The .xlsx file can be generated with formatting and images. A template file with the extension .xslx must be created with a software like Microsoft Excel or Google Sheets.

Limits and fair use

  • The limitation of the input file size is 500 Mb.
  • The limitation of the files that can be generated is 500.
  • The limitation of the sheets per file is 25.

Even if there is these limitations you can encouter errors on your files with metrics below of the limitations above. This is because your generated file can be very different that the ones we used for our tests.
In addition to these limitations, we encourage you to apply these best practices:

  • If you use images, use images with small presets.
  • Do not reference images or files that are not stored in your Product-Live account.
  • Be nice with Excel. Excel is not a database, so don't try to export all your items in an Excel file. If you need to export all of your data, use a format like CSV or XML or JSON.

FAQ

How to output multiple files?

Use as many <File> block as you need.

Why this is not the same format than the XLSX to XML action?

This is not the same use. When you do a XLSX to XML transformation you usually need to read a table of data containing a list of records. When you want to write in a .xlsx file, it will be much easier in you transform action to use cells/cell instead of rows/row because you will not always write a table, but sometimes just some cells.

Problem with numbers like EAN 13

In your Excel template you need to format your column as text.

Problems with formatting

Ensure that you have formating all your cells the same way.

How can I chain with the Zip action?

There is two methods: either you want to create a Zip where all Excel files will be at the root of the archive, then you can directly chain the Zip task, nor you want a personalized Zip archive then you have to chain first with a Transform XSLT and then the Zip task.

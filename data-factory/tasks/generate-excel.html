Product-Live Data Factory

Generate Excel

Description

You need to to generate one or multiple .xlsx file. The .xlsx file can be generated with formatting and images. A template file with the extension .xslx must be created with a software like Microsoft Excel or Google Sheets.

Inputs

ParamDescription
InputYou can select between Output or File.Select Output if you want to use the output of another action.Select File if you to use always the same file.
TemplateYou must select a .xlsx file.


Expected input

XPathDescription
Generate-ExcelRoot of the document
./FileFor each files you want to generate.You can generate multiple files.
./File/File-NameFile name
./SheetsSheets block
./Sheets/SheetFor each sheet you want to generate
./Sheets/Sheet/Sheet-NameName of the sheet.Either the sheet already exists and it writes in it,or it does not exist and it is created.
./Sheets/Sheet/CellsCells block
./Sheets/Sheet/Cells/CellContent of the cell. For IMAGE type cells put the original url. Always put a CDATA.
../Cell/@lineCoordinate of the line. The index starts at 1.
../Cell/@columnCoordinate of the column. The index starts at 1.
../Cell/@typeSet type="IMAGE" if you want to output an image in the cell. The image will be automtically resized to fit the cell.
../Cell/@urlOnly when you write a link. A valid url
../Cell/@link-typeURL | FILE | EMAIL | DOCUMENT
URL : open a valid url on your default web browser
FILE: open a file on your local system , the url attribute must be a valid path
EMAIL: open your default email program, the url attribute must be a valid email
DOCUMENT: reference another cell in the document, the url attribute must be like : 'Sheet1'!A1
